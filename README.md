# ActStimRL

- The current repository is for a project named " Reinforcement learning in Parkinson's disease: Dissociating Action- and Color-based learning", conducted at Danish Research Centre for Magnetic Resonance, Department of Radiology and Nuclear Medicine, Copenhagen University Hospital – Amager and Hvidovre, Copenhagen, Denmark
- This study aimed to determine the extent to which neurodegeneration in Parkinson’s disease affects action-reward and color-reward associations and whether dopamine replacement via L-DOPA can mitigate this deficit.
